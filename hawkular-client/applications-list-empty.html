Steps to take after downloading:

Place a file in a target directory

The default location is the current working directory, unless you specified a directory in previous steps. Note: storing the installer in the same directory as your server instance, will simplify the command line needed to run the installer (as you won't need to provide a -targetlocation-).

Run the installer

In a CLI tool, run java jar hawkularwildfly-agent-installer.jar -encryptionkey your_key. For detailed installation instructions, refer to link to docs.
